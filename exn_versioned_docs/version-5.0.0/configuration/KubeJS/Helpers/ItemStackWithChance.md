# ItemStackWithChance

---

## Object Structure
```javascript
{
    chance: number,
    count: number,
    item: string
}
```

## Fields

`chance` - Value between 0.0 and 1.0
`count` - Whole number greater than 0
`item` - Item to drop

## Example

```js
{
    chance: 0.75,
    count: 20,
    item: 'minecraft:melon_slice'
}
```
